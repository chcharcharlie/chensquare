FirstSolo

A flight training platform built for CFIs and student pilots — track lessons, log maneuvers, and work toward that first solo flight.

Open FirstSolo.app ↗

What is this?

FirstSolo is a web app for flight training — built with AI from scratch in a few days. A CFI (Certified Flight Instructor) can create a profile, invite their students, and track lesson-by-lesson progress toward the student's first solo.

Students log in to see their own progress, upcoming lesson goals, and what maneuvers still need sign-off. Everything's in one place, no paper syllabus required.

Features

👨‍✈️
CFI Dashboard
Manage students, track lesson history, and monitor overall progress from one view.
📋
Lesson Logging
Log each lesson with maneuvers practiced, notes, and sign-offs. Linked to a standard syllabus.
📈
Student Progress
Students can see exactly where they stand — what's done, what's left, and what's next.
✅
Solo Readiness
Track checklist items and requirements for solo endorsement, stage by stage.

Built with

Firebase (Auth + Firestore), React, and a lot of AI-assisted coding. Built over a few sessions as a vibe-coding experiment.
